## HimaCanvas

**HimaCanvas** は、ブラウザ上で動作する軽量なお絵かきツールです。

### 特徴

- マルチレイヤー対応
- 20種類以上のブラシ
- 色調補正フィルター
- ショートカットキー対応
- PSD / PNG エクスポート

### 技術スタック

- HTML5 Canvas API
- WebAssembly (WASM)
- IndexedDB
