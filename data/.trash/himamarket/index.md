## HimaMarket

**HimaMarket** は、暇人技術部が開発したオープンソースのフリマアプリケーションです。

### 特徴

- 直感的な出品フロー
- リアルタイムチャット機能
- 評価・レビューシステム
- カテゴリ別検索

### 技術スタック

- React Native
- Cloudflare Workers
- D1 Database
- R2 Object Storage
